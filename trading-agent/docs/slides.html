Our Solution

The Trust Triangle

Every trade must satisfy all three — simultaneously.

1. Identity

ERC-8004 on-chain identity. Agent registered as ERC-721 NFT. Discoverable via Agent Card.

2. Constraints

Risk Router contract enforces 7 gates. The AI proposes — the system decides. No bypass possible.

3. Auditability

Every intent signed with EIP-712. Validation artifacts hashed on-chain. Cryptographic proof, not logging.
